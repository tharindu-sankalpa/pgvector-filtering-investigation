① Enter Top Layer
Fixed entry node A
One predefined entry node. Top layer is never fully scanned. Entry point is fixed — not chosen based on proximity to Q.
② Greedy Hop
Jump toward Q
Check neighbors. Hop to whichever is closer to Q. Repeat until no neighbor is closer — local minimum reached. Then descend.
③ Descend & Repeat
Drop to same node below
At local min, drop to the exact same node in the layer below. New finer-grained neighbors appear. Resume greedy hopping.
④ Bottom Layer Search
Collect top-k results
Now in layer 0 with all vectors. Explore ef_search candidates. Rank all by distance to Q. Return the k closest — not just one.
